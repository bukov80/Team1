## 🚨 Пріоритетні додаткові завдання

### 👩‍💼 Oksana — HR-аналітика
HR-відділ повідомив про підозрілу аномалію: у деяких співробітників бонус більший за базову зарплату.
* **Необхідно:**
  * Знайти всіх співробітників, де `bonus > base_salary`.
  * Побудувати scatter plot (base_salary vs bonus) з лінією $y = x$.
  * Порахувати середній бонус по регіонах та посадах (`title`).
  * Перевірити, чи є співробітники з однаковою посадою, але суттєво різними зарплатами.
* **Файл:** `Oksi_HR.py`

### 📊 Iryna — Аналіз промоакцій
Шеф хоче один простий графік для презентації на раді директорів: *«Які промоакції дають найбільший обсяг продажів?»*.
* **Необхідно:**
  * Згрупувати `order_items` по `promotion_id`.
  * Порахувати сумарний revenue та сумарний discount.
  * Об'єднати результат з таблицею `promotions`.
  * Побудувати horizontal bar chart для ТОП-5 промоакцій за revenue.
  * Відобразити `discount_pct` кольором.
  * Вивести таблицю з назвою акції, revenue та знижкою.
* **Файл:** `Кріт-2 Шеф хоче один простий графік.py`

### 🕵️‍♂️ Serhii — Аналіз аномальної активності клієнта
Система зафіксувала клієнта, який зробив аномально багато замовлень за один день.
* **Необхідно:**
  * Знайти клієнта або клієнтів з найбільшою кількістю замовлень за один день.
  * Визначити кількість замовлень, загальну суму, товари та спосіб оплати.
  * Перевірити наявність повернень та причини повернення.
  * Побудувати графік історії замовлень по днях.
  * Зробити висновок: оптовий клієнт чи потенційний фрод.
* **Файл:** `anomalous_daily_customer_orders.py`
