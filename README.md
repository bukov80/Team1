## Додаткові завдання (червона мітка)

### Oksana — HR-аналітика

HR-відділ повідомив про підозрілу аномалію — у деяких співробітників бонус більший за базову зарплату.

Необхідно:

1. Знайти всіх співробітників, де `bonus > base_salary`
2. Побудувати scatter plot (`base_salary` vs `bonus`) з лінією `y = x`
3. Порахувати середній бонус по регіонах та посадах (`title`)
4. Перевірити, чи є співробітники з однаковою посадою, але суттєво різними зарплатами

Файл:
`Oksi_HR.py`

---

### Iryna — Аналіз промоакцій

Шеф хоче один простий графік для презентації на раді директорів: «Які промоакції дають найбільший обсяг продажів?»

Необхідно:

1. Згрупувати `order_items` по `promotion_id`
2. Порахувати сумарний `revenue` та сумарний `discount`
3. Об'єднати результат з таблицею `promotions`
4. Побудувати horizontal bar chart для ТОП-5 промоакцій за revenue
5. Відобразити `discount_pct` кольором
6. Вивести таблицю з назвою акції, revenue та знижкою

Файл:
`Кріт-2 Шеф хоче один простий графік.py`

### Serhii — Аналіз аномальної активності клієнта

Система зафіксувала клієнта, який зробив аномально багато замовлень за один день.

Необхідно:

1. Знайти клієнта (або клієнтів) з найбільшою кількістю замовлень за один день
2. Визначити:

   * кількість замовлень;
   * загальну суму замовлень;
   * перелік товарів;
   * спосіб оплати;
3. Перевірити наявність повернень та причини повернення
4. Побудувати графік історії замовлень по днях
5. Зробити висновок: оптовий клієнт чи потенційний фрод

Файл:
`anomalous_daily_customer_orders.py`
