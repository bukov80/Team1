# 📊 Проєкт з Data Analytics — Team 1

**Комплексний аналіз даних інтернет-магазину** на базі SQLite.  
Проєкт охоплює HR-аналітику, маркетинг, продуктову аналітику, виявлення аномалій, сезонність продажів та антифрод-моніторинг.

---

## 📂 Структура проєкту

- **`online_store.db`** — основна база даних SQLite
- **`main.py`** — головний інтегрований скрипт (об’єднує всі завдання)
- **`README.md`** — цей файл (повний опис проєкту)
- **`requirements.txt`** — список необхідних бібліотек
- **`scripts/`** — папка з індивідуальними скриптами учасників

## 🛠 Технології

- **Python 3.8+**
- **Pandas**, **NumPy** — обробка даних
- **Matplotlib**, **Seaborn** — візуалізація
- **NetworkX** — графовий аналіз
- **SQLite3** — робота з базою даних
