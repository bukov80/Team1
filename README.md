## 🚨 Пріоритетні додаткові завдання

### 👩‍💼 Oksana — HR-аналітика
HR-відділ повідомив про підозрілу аномалію: у деяких співробітників бонус більший за базову зарплату.
* **Що зроблено:**
  * Знайдено всіх співробітників, у яких `bonus > base_salary`.
  * Побудовано scatter plot (base_salary vs bonus) з лінією $y = x$.
  * Розраховано середній бонус за регіонами та посадами (`title`).
  * Проведено перевірку на предмет суттєвої різниці в зарплатах на однакових посадах (потенційна дискримінація).
* **Файл:** `Oksi_HR.py`

### 📊 Iryna — Аналіз промоакцій для керівництва
Підготовка графіку для презентації на раді директорів: *«Які промоакції дають найбільший обсяг продажів?»*.
* **Що зроблено:**
  * Агреговано дані `order_items` за `promotion_id` (розрахунок сумарного revenue та discount).
  * Об'єднано результати з довідником `promotions`.
  * Візуалізовано горизонтальний bar chart для ТОП-5 промоакцій за виручкою, де колір відображає відсоток знижки (`discount_pct`).
  * Сформовано підсумкову таблицю (назва акції, revenue, знижка).
* **Файл:** `Promo_Analysis_Boss.py`

### 🕵️‍♂️ Serhii — Антифрод та аналіз аномальної активності
Система зафіксувала клієнта, який зробив аномально багато замовлень за один день.
* **Що зроблено:**
  * Ідентифіковано клієнтів із піковою добовою кількістю замовлень.
  * Деталізовано профайл аномалії: кількість замовлень, сума чеку, перелік товарів, методи оплати.
  * Перевірено історію повернень та їхні причини.
  * Візуалізовано динаміку замовлень клієнта по днях.
  * Сформульовано фінальний висновок: це великий оптовий покупець чи потенційний фрод.
* **Файл:** `Anomalous_Customer_Orders.py`
